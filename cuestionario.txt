4. Describe el procedimiento por el cual una canción de Spotify llega del servidor de la compañía a tu celular.

Mediante TCP por sus siglas en inglés, (Transmission Control Protocol). Cuando solicitas una canción al dispositivo,
Spotify envía la canción en muchos paquetes, cuando estos llegan TCP hace un inventario y envía una contestación a
cada paquete recibido,si todos están ahí, TCP los envía y listo. Sin embargo si TCP no encuentra los paquetes correctos
no los enviará y la canción se escuchará mal o faltarán partes, afortunadamente Spotify los puede reenviar para que
la canción suene correctamente. <!-- ---- 6oct | @SRTP: ok ----  -->

6. En dos párrafos cuéntanos lo que aprendiste sobre encriptación y llaves públicas.

El cifrado o encriptación, es un proceso en el cual la información se revuelve o se cambia para ocultar el 
mensaje original. Por otro lado, el decifrado es el proceso de ordenar un mensaje para que sea legible.

Las llaves públicas o llaves de cifrado asimétrico, pueden ser intercambiadas con cualquiera, o con una llave
privada (no compartida).
Las llaves públicas sirven para cifrar datos y la puede utilizar quien sea para crear
mensajes secretos, pero este solo puede ser descifrado por una computadora con acceso a una llave privada. <!-- ---- 6oct | @SRTP: ok ----  -->

8. ¿Cómo funciona la búsqueda de google?

El Internet es una red de páginas conectadas a través de hipervínculos, los buscadores constantemente corren un 
programa llamado la "araña" que camina a trevés de estas páginas para recolectar información sobre ellas. Cada 
vez que encuentra un hipervínculo, lo sigue hasta que haya visitado cada página que pueda encontrar en todo el 
Internet. Por cada página que visita la "araña" graba información que pudiera necesitar para una búsqueda, pero
la añade a una base de datos llamada índice de busqueda. <!-- ---- 6oct | @SRTP: ok ----  -->
